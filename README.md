# HTML & CSS Practice Projects

This repository contains a collection of small **HTML and CSS projects** that I created during **college sessions**.
These exercises were done to practice the basics of **web page structure, styling, and layout design**.

## Technologies Used

* HTML5
* CSS3

## What These Projects Include

* Basic webpage layouts
* Use of headings, links, images, and forms
* Simple styling using CSS
* Practice with page structure and UI elements

## Purpose

The main goal of these projects was to **build a strong foundation in frontend web development** by applying concepts learned in class.

## How to Run

1. Download or clone the repository
2. Open any `.html` file in a web browser

These projects are simple practice exercises, but they helped me understand the **fundamentals of building and styling webpages**.
